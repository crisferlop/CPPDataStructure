Descripción detallada

template<typename VALUE>
class BNode< VALUE >

Clase BNode. representa un nodo del arbol B, es una de las hojas o ramas del mismo contiene una lista de datos ordenados.

El construtor de BNode es elegante, pues agrega el valor inicial al arreglo de datos, la misma inicializa la cantidad de datos que contendra.

Parámetros
